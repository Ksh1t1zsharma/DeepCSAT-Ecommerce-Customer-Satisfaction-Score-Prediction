# Contributing to DeepCSAT

Thanks for your interest in contributing! Please follow these steps:

1. Fork the repository and create a new branch for your feature or bugfix.
2. Ensure code is well-documented and includes tests where applicable.
3. Run linters and formatters before submitting a PR.
4. Open a pull request with a clear description of changes and why they are needed.

We welcome contributions that improve documentation, add features, fix bugs, or enhance reproducibility.
